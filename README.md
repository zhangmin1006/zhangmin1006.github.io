# zhangmin1006.github.io

This repository contains the static personal website for Min Zhang, built from the provided CV document.

## Website files

- `index.html` — main website page
- `styles.css` — styling for the site

## Local usage

1. Open `index.html` in a browser.
2. Or use a static server such as `Live Server` in VS Code.

## GitHub Pages

This repo is intended to be published as a GitHub Pages site at `https://zhangmin1006.github.io`.

To push this repository to GitHub:

```bash
cd zhangmin1006.github.io
git init
git add .
git commit -m "Initial personal website"
git remote add origin https://github.com/zhangmin1006/zhangmin1006.github.io.git
git push -u origin main
```
